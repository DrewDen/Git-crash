  • If you want to add files to your repository you can use the command git add [file your adding]in git bash
  • Always use git status. This will show the changes you made.
  • If you mad a mistake on adding a file using git rm--cached[file your removing]is the command to use to remove the file
  • If you want to add all your files you could use git add.and this will add all the changes you made to your files. Also I believe using the * key will also work
  • Next you can commit your files using git commit. If you use this metthod two options can come into play:
    • 1. Your code editor will pop up and a seprate widow will appear letting you write in a comment.
    • 2. You can usegit commit -m'whatever you want to put between this two parentheses'and this will skip you code editor opening up.
